# Прогноз погоды

**Боровский Максим Олегович · 367916 · К3344**

Небольшой учебный проект (чистый HTML/CSS/JS). Веб-приложение прогноза погоды с определением текущего местоположения, добавлением нескольких городов, обновлением данных по кнопке и сохранением выбранных локаций в localStorage.

## Функционал
Функционал
- При первом открытии запрашивается доступ к геопозиции и показывается прогноз на сегодня + 2 дня для “Текущее местоположение”.
- При отказе в геолокации отображается форма добавления города для получения прогноза.
- Возможность добавить как минимум 2 дополнительных города и переключаться между локациями.
- Кнопка "Обновить" выполняет повторные HTTP-запросы к API (без перезагрузки страницы).
- Отображение состояний интерфейса: загрузка / ошибка / успех.
- При добавлении города есть выпадающий список подсказок, можно выбрать только существующий город; при некорректном вводе показывается ошибка под полем.
- Сохранение состояния: выбранная локация и список городов сохраняются в localStorage и восстанавливаются после перезагрузки (при этом прогноз запрашивается заново).
